Tujuan Pembelajaran

Setelah menyelesaikan bab ini, mahasiswa diharapkan mampu:

  1. Memahami pengertian dan fungsi mikrokontroler
  2. Mengenal arsitektur dasar mikrokontroler (CPU, memori, I/O)
  3. Membedakan mikrokontroler dengan mikroprosesor
  4. Menyebutkan contoh penerapan mikrokontroler dalam kehidupan sehari-hari

Apa Itu Mikrokontroler?

Mikrokontroler adalah sebuah komputer kecil (microcomputer) yang terintegrasi dalam satu chip IC. Di dalamnya terdapat:

  • CPU (Central Processing Unit) — otak pemrosesan
  • RAM (Random Access Memory) — penyimpanan sementara
  • Flash Memory — penyimpanan program
  • I/O Ports — pin untuk berinteraksi dengan dunia luar
  • Timer/Counter — untuk fungsi pewaktu
  • ADC/DAC — konversi sinyal analog-digital

Mikrokontroler vs Mikroprosesor

Aspek Mikrokontroler Mikroprosesor
Memori Terintegrasi Eksternal
I/O Terintegrasi Memerlukan chip tambahan
Harga Murah Lebih mahal
Konsumsi daya Rendah Tinggi
Contoh Arduino, ESP32, STM32 Intel i7, AMD Ryzen

Contoh Penerapan

  • Otomotif: Sistem ABS, airbag, engine control unit
  • IoT: Smart home, sensor monitoring
  • Robotika: Kontrol motor, navigasi
  • Medis: Alat monitoring detak jantung
