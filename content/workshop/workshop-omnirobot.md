Halo, Praktikan! Siap untuk mulai merakit dan mengendalikan robotmu sendiri dari layar kaca? Workshop Raspberry Pi (Robot) dari PSMURO ini dirancang khusus agar kamu bisa langsung praktik (*hands-on*) membuat *Mobile Robot* 4WD yang dapat dikendalikan dan menyiarkan video secara langsung melalui *web*.

> **"Tingkatkan kemampuan teknologimu! Mari pelajari cara membangun sistem kendali robot jarak jauh menggunakan Web Framework dan Video Streaming."**

---

## 🎯 Apa Saja Aktivitas di Workshop Ini?

Kita akan belajar membangun sistem kendali robot berbasis *web* dari dasar hingga tahap integrasi kamera. Berikut rinciannya:

| Aktivitas | Detail Pembelajaran |
| :--- | :--- |
| 🌐 **Pemrograman Web dengan Flask** | Memahami *microweb framework* **Flask** pada Python 3 untuk membangun aplikasi *web* yang ringan dan fleksibel. |
| ⚙️ **Komunikasi Hardware** | Memahami prosedur komunikasi Raspberry Pi dengan **H-Bridge Motor Driver (L298N)** dan **Motor DC** untuk menggerakkan roda robot. |
| 🎥 **Integrasi Kamera & Streaming** | Mengkonfigurasi **WebCam** dan perangkat lunak **MJPG Streamer** untuk menangkap dan menyiarkan video secara *real-time* dengan format *Motion JPEG*. |
| 💻 **Kendali Navigasi HTML** | Merancang *User Interface* (UI) berbasis **HTML** yang terintegrasi dengan Python untuk mengontrol arah gerak robot (Maju, Mundur, Kiri, Kanan, Berhenti) dari *browser*. |
