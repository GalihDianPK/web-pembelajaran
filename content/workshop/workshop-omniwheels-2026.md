Halo, Praktikan! Tertarik menggabungkan kecerdasan buatan dengan robotika? Workshop Raspberry Pi (Omniwheels) dari PSMURO ini dirancang khusus agar kamu bisa langsung praktik (*hands-on*) merakit robot yang bisa bergerak ke segala arah dan mengenal wajah manusia!

> **"Tingkatkan kemampuan teknologimu! Mari pelajari cara membangun robot cerdas berbasis Computer Vision dari nol hingga terkoneksi ke antarmuka Web."**

---

## 🎯 Apa Saja Aktivitas di Workshop Ini?

Kita akan belajar membangun sistem robotika cerdas mulai dari pengontrolan *hardware* hingga implementasi *machine learning* untuk pengenalan wajah. Berikut rinciannya:

| Aktivitas | Detail Pembelajaran |
| :--- | :--- |
| 🤖 **Pengenalan Hardware Omniwheels** | Memahami struktur pergerakan robot *Omniwheels*, penggunaan **Motor Driver IBT2**, dan **Baterai LiPo**. |
| 🌐 **Pemrograman Web Navigasi** | Membangun antarmuka kontrol (GUI) berbasis web menggunakan **HTML, CSS, JavaScript, Bootstrap**, dan **jQuery**. |
| 📸 **Akuisisi & Training Dataset** | Menggunakan WebCam untuk mendeteksi fitur wajah dan mengumpulkan dataset gambar menggunakan pustaka **OpenCV** dan **PIL (Pillow)**. |
| 🧠 **Algoritma Face Recognition** | Mempelajari dan menerapkan algoritma pengenalan wajah **Local Binary Pattern Histogram (LBPH)**. |
| ⚙️ **Threading & Integrasi Python** | Menerapkan teknik *Threading* pada Python 3 agar Raspberry Pi dapat menjalankan kontrol motor web dan streaming kamera (*multitasking*) secara bersamaan. |
