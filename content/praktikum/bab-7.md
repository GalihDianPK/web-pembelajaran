## Tujuan Pembelajaran

1. Mengenal software perancangan PCB (KiCad)
2. Membuat skematik rangkaian elektronika
3. Merancang layout PCB
4. Memahami proses fabrikasi PCB

## Tahapan Desain PCB

1. **Skematik** — Menggambar rangkaian elektronika
2. **Footprint Assignment** — Menentukan bentuk fisik komponen
3. **Layout PCB** — Menempatkan komponen dan routing jalur
4. **Design Rule Check** — Verifikasi aturan desain
5. **Export Gerber** — File untuk produksi

## Latihan

1. Buat skematik rangkaian LED blinking di KiCad
2. Rancang layout PCB sederhana (single layer)
3. Export file Gerber dan periksa menggunakan Gerber viewer
