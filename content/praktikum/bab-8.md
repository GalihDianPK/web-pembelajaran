## Tujuan Pembelajaran

1. Mengintegrasikan seluruh pengetahuan dari Bab 1-7
2. Merancang dan membangun proyek embedded system
3. Mendokumentasikan proyek secara profesional
4. Mempresentasikan hasil karya

## Panduan Proyek Akhir

### Persyaratan

- Proyek harus menggunakan **minimal 2 sensor** dan **1 aktuator**
- Menggunakan komunikasi WiFi atau Bluetooth
- Memiliki **user interface** (LCD, web dashboard, atau aplikasi mobile)
- Didokumentasikan dalam laporan tertulis

### Tema Proyek (Pilih Salah Satu)

1. **Smart Home** — Sistem otomatisasi rumah
2. **Environmental Monitoring** — Pemantauan lingkungan
3. **Robotika** — Robot dengan kontrol jarak jauh
4. **Wearable Device** — Perangkat yang dipakai di tubuh
5. **Tema Bebas** — Dengan persetujuan dosen pembimbing

### Timeline

| Minggu | Kegiatan |
|--------|----------|
| 1-2 | Perencanaan dan desain sistem |
| 3-4 | Pembuatan prototype hardware |
| 5-6 | Pemrograman dan integrasi |
| 7 | Pengujian dan debugging |
| 8 | Presentasi dan demo |

## Format Laporan

1. Latar belakang dan tujuan
2. Diagram blok sistem
3. Skematik rangkaian
4. Source code (dengan penjelasan)
5. Hasil pengujian
6. Kesimpulan
